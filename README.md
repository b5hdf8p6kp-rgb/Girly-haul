# Girly Haul — Static Starter Site

This is a simple, responsive, static site for "Girly Haul" built with plain HTML/CSS and vanilla JavaScript.

Features:
- Home, Shop, Product detail, About, Contact pages
- Product data in `products.json`
- Client-side cart saved in localStorage (demo checkout)
- Responsive, pastel-themed styling
- Placeholder images (Unsplash). Replace with your own assets.

How to deploy (GitHub Pages)
1. Create a new GitHub repository and push these files to the default branch (usually `main`).
2. In the repo Settings → Pages, set the Source to the default branch root and save.
3. Visit the provided github.io URL after a minute.

Customize
- Replace `products.json` with your real products (same schema: id, name, price, image, description).
- Update `contact.html` form `action` to your Formspree endpoint or server endpoint.
- Replace images with your CDN or `/assets/` files.
- For real payments: integrate Stripe Checkout, Snipcart, Shopify, or a server-side solution.
